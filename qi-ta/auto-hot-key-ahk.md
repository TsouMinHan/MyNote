# Auto Hot Key （AHK）

AHK 本身程式不會做任何事情，需要透過腳本（檔名為 `ahk`）來告訴程式該做些什麼事情。  
點擊腳本檔案執行。  
AHK.exe 設定路徑，執行腳本。

右鍵單擊托盤圖標以顯示托盤菜單, 默認情況下具有以下選項:

* Open - \(打開\) -打開腳本的[主窗口](https://wyagd001.github.io/zh-cn/docs/Program.htm#main-window).
* Help - \(幫助\) - 打開AutoHotkey 離線幫助文件.
* Window Spy - \(窗口監視器\) - 顯示關於窗口的各種信息.
* Reload This Script - \(重載這個腳本\) -請參閱[Reload](https://wyagd001.github.io/zh-cn/docs/commands/Reload.htm) .
* Edit This Script - \(編輯這個腳本\) -請參閱[Edit](https://wyagd001.github.io/zh-cn/docs/commands/Edit.htm) .
* Suspend Hotkeys - \(掛起熱鍵\) -[掛起](https://wyagd001.github.io/zh-cn/docs/commands/Suspend.htm)或取消掛起熱鍵.
* Pause Script - \(暫停腳本\) -[暫停](https://wyagd001.github.io/zh-cn/docs/commands/Pause.htm)或取消暫停腳本.
* Exit - \(退出\) - 退出腳本.
