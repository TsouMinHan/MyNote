# VSCode 快捷鍵





### 一般

#### Command Palette\(中文我不知道是甚麼\)

根據當前上下文訪問所有可用命令。這是[VS Code Tips and Tricks](https://github.com/Microsoft/vscode-tips-and-tricks)所敘述的。 我自己目前只有在選擇Python編譯器及使用Jupyter時會使用到。

#### 快速開啟檔案

`ctrl` + `p` 可以開啟最近開過的檔案，在開發Flask專案時應該不錯用記住這個指令吧。

### 編輯

#### 剪下

`ctrl` + `x` 直接剪下一整行，原本的內容會不見唷。

#### 複製

`ctrl` + `c` 直接複製一整行，原本的內容會留著。

#### 上下移動程式碼

`alt` + `↑`/`↓` 移動程式碼，超方便。

#### 左右移動程式碼

`ctrl` + `[`/`]` 移動程式碼，超方便，但是要注意如果是用中文輸入法沒辦法使用，要用英文輸入法。

#### 新增行

`ctrl` + `enter` 往下新增一行，游標不用在最尾端。 `ctrl` + `shift` + `enter` 往上新增一行，游標不用在最尾端。

#### 顯示成對的括弧位置

`ctrl` + `shift` + `\` 可以用來確認括弧是否成對，感覺在寫JavaSctipt時會很方便，就不用一直看到眼睛脫窗了，新發現！

#### 縮小/展開程式碼區塊

`ctrl` + `k` `ctrl` + `j` 展開所有程式碼區塊。 `ctrl` + `k` `ctrl` + `0` 縮小所有程式碼區塊。

#### 切換註釋行

`ctrl` + `/` 切換註解行，有註解&gt;沒註解，沒註解&gt;有註解。

#### 切換註釋區塊

`shift` + `alt` + `a` 切換整個註解區塊，有註解&gt;沒註解，沒註解&gt;有註解。

### 導航

#### 跳到第N行

`ctrl` + `g`

#### 切換已開啟的檔案

`alt` + `→`/`←` 開很多檔案時可以使用。

### 搜尋和取代

#### 搜尋

`ctrl` + `f` 搜尋，可以選取要搜尋的字之後再按。

#### 取代

`ctrl` + `h` 取代，可以選取要搜尋的字之後再按。

#### 往前/往後搜尋

`f3`/`shift` + `f3` 往後/往前搜尋，這樣要往前找的時後就不用一直用滑鼠點了。

#### 選取所有查詢的結果

`alt` + `enter` 搜尋之後用這個快捷鍵會在所有查詢到的結果後面出現游標，可以用來同時編輯所有結果。

### 游標相關

#### 多個游標

`alt` + click 就產生多的游標，同時修改多行時可以使用。

#### 選取此行

`ctrl` + `l` 直接選取整行，感覺滿方便的，應該用的到啦。

#### 選取所有符合選取內容的字

`ctrl` + `shift` + `l` 跟`alt` + `enter`類似，但是是選取整行或一個段落。

#### 重新命名變數

`f2`

#### 選取所有出現的字串

`ctrl` + `f2` 例如想改變同一個變數的名稱時可以使用，但是要注意假設要修改a，其他內容包含a的也會被修改，所以要小心。

#### 向外/向內選取

`ahift` + `alt` + `→` 向外選取，在選取變數時應該很方便。 `ahift` + `alt` + `←` 向內選取。

### 程式碼編輯

#### 顯示變數/方法/物件相關資訊

按著 `ctrl` 滑鼠指向方法或物件。

#### 自動排版

`ctrl` + `alt` + `f`

#### 轉到變數/方法/物件定義處

`f12` 游標直接指向該行 `alt` + `f12` 另外跑出一個小框框顯示。

#### 顯示使用該變數/方法/物件的地方

`shift` + `f12` 跟 `alt` + `f12` 差不多，但是會顯示在哪幾行使用該變數/方法/物件。
