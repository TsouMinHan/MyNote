# Day 4：Array Cardio Day 1

`console.table()`：將資料以 table 的方時呈現，比較好閱讀，原來有這樣的指令，第一次知道呢...，以後看 Array 的資料就可以使用這個了

`sort()`：因為資料的筆數不同分為**穩定排序**與**非穩定排序**。

穩定排序：每次排出來都一樣，Chrome 採用這樣的方法，所以在 Chrome 上執行 JavaScript 的 `sort()` 是比較沒問題的。

非穩定排序：因資料差異會導致結果不一樣，例如將 10 筆資料中的其中 1 筆刪除掉，會導致順序整個不同，因此之前的人傾向自己寫排序法。
