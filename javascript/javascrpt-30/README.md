# JavaScrpt 30

## Day 7：Array Cardio Day 2

陣列的一些指令

```javascript
array.some( arr =>
     return "some conditional sentence"   //有符合就回傳 true
}

array.every( arr =>
     return "some conditional sentence"   //全有符合就回傳 true
}

array.find( arr =>
     return "some conditional sentence"   //回傳符合條件的第一筆資料
}

array.findIndex( arr =>
     return "some conditional sentence"   //回傳符合條件的第一筆資料的index
}

// 從 index 開始動0筆資料, 第三個參數沒有就是刪除，有的話就是新增，會動到原始資料
array.splice(index, 0) 

// 從 start 取資料至 end，不會動到原始資料
array.slice(start, end)


```
