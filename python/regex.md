# 正規化

正則表達式樣式跟被搜尋的字串可以是 Unicode 或 bytes，但不能混合使用。 在字串中反斜線 `\` 代表允許使用特殊符號，例如要搜尋 `\` 符號，正則表達式要打 `\\`，或者使用 `r` 當前綴符號就不會對 `\` 做任何處理，因此 `r"\n"` 意思就是 ```\``n``` 兩個字。

## 正則表達式樣式符號介紹

`.`：任何字符   
`^`：特定字符開頭   
`$`：特定字符做結尾   
`*`：特定字符出現 0 到任意次   
`+`：特定字符出現 1 到任意次   
`?`：特定字符出現 0 到 1 次   
`{m}`：特定字符出現 m 次   
`{m, n}`：特定字符出現 m 到 n 次   
`[]`：集合內所有的字符，\[a-z\] 搜尋任何小寫的字，\[0-5\]\[0-9\] 搜尋任何 00~59 的兩位數數字，\[0-9A-Fa-f\] 搜尋十六進制數字，\[\(+\*\)\] 只會搜尋這幾個文法字符`(`、`+`、`*`、`)`，將搜尋除了 5 以外的所有字符   
`|`：搜尋 A 或 B 字符   
`\d`：搜尋數字 0-9   
`\D`：搜尋非數字  
`\s`：搜尋空白符號，包括`\n`   
`\S`：搜尋非空白符號   
`\w`：搜尋任何字符，包括數字   
`\W`：搜尋任何非字符，包括空白符號

## search

掃描整個字串找到符合正則表達式樣式的第一個位置，並返回一個物件（顯示起始位址與結束位址），如果搜尋不到，就返回一個`None。`

## match

從頭開始掃描整個字串找出符合正則表達式樣式的位置，就返回一個相應的物件（顯示起始位址與結束位址），如果搜尋不到，就返回一個`None。`

## fullmatch

跟 match 相似，但必須要整個字串完全相符才會回傳結果，否則為`None。`

## findall

找到字串中符合正則表達式樣式的字串，並回傳所有結果，若無結果則回傳空串列。
