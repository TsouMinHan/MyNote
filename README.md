# README

從 Hexo 移植過來了，花了好大的力氣阿，覺得要自己手動部署到 Github 上太麻煩了，且要在其他電腦編輯的話太麻煩了，環境要先裝好。改用 Gitbook 比較輕鬆，但是 Gitbook 無法 export 筆記內容，差點又讓我放棄他了，好險最後可以同步到 Github 上，這樣有需要時就可以下載下來了，Gitbook也有個缺點就是他的排版，明明有空格卻會自動消除..真的是很討厭。

最後決定的方式就是拉下來修改後再 push 到 Github 維持統一性。

基本上還是用 Gitbook 編輯文章好了，自己在本地端寫文章，修改檔名等等，導致 Gitbook這邊路徑錯誤，花了好多時間在找復原方法，不知為什麼Gitbook跟Github同步上會有些不同，最後在Gitbook這邊Restore回去正常的版本，反正就是乖乖在這邊編輯就好，省的發生更多麻煩。
